## Kicad pcb editor

The PCB was created with [kicad](https://www.kicad.org/). 

```sh
sudo apt-get install kicad
```

#### Export as SVG

- Select only `F.Cu`
- Set SVG Page Size to `Board area only`

![alt text](encoder2-F_Cu.svg)

## SVG to Gcode

To convert SVG to Gcode, use https://cam.openbuilds.com/
